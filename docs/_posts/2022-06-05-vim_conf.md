## 基本配置
```
# _vimrc
colorscheme desert
set nu
set noswapfile
set nobackup

set ts=4
set expandtab
```

## 快捷键备忘
- Ctrl + a ：移到命令行首
- Ctrl + e ：移到命令行尾
- Ctrl + d ：删除光标处的字符
- Ctrl + r：逆向搜索命令历史
- Ctrl + u ：从光标处删除至命令行首
- Ctrl + k ：从光标处删除至命令行尾
- Ctrl + w ：从光标处删除至字首
- Alt + d ：从光标处删除至字尾
