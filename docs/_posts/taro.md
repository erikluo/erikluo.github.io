## Taro
开放式跨端跨框架解决方案，支持使用 React/Vue/Nerv 等框架来开发微信/京东/百度/支付宝/字节跳动/ QQ 小程序/H5/React Native 等应用。
* [github](https://github.com/NervJS/taro)
* [docs](https://docs.taro.zone/docs/)

* [一款基于 Taro 框架开发的多端 UI 组件库](https://github.com/NervJS/taro-ui)
* [首个 Taro 多端统一实例 - 网易严选（小程序 + H5 + React Native） - By 趣店 FED](https://github.com/js-newbee/taro-yanxuan)
